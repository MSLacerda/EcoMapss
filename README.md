# EcoMapss
Ecomapss app created by a students of IFCE Campus Crato, which helps who is want learn about flora and fauna.

## Requirements

* [NodeJS]
* [NPM]


## Tutorial

Clonig & Installing the repo: 

``` 
git clone https://github.com/MSLacerda/EcoMapss 

cd EcoMapss/app

npm install 
```
or
```
yarn install
```
